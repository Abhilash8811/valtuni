03 / How to Play

Rules in 60 seconds

01

Classic Sudoku Rules Apply

Fill the 9×9 grid so every row, column, and 3×3 box contains the digits 1–9 exactly once.

02

Cages Must Sum to Their Target

Each dashed cage shows a number in the top-left corner. The digits in that cage must add up to exactly that number — with no repeats inside the cage.

03

Tap a Cell, Tap a Number

Select a cell then tap 1–9 to fill it. Switch to Notes mode to pencil in candidates. Long-press any cage cell to see how much it still needs.

04

No Feedback Until the Grid is Full

The puzzle only validates when every cell is filled. Use logic, not trial-and-error. Undo/Redo are always available.

05

Beat Your Time for More Stars

Solve fast enough to earn 3 stars. Stuck? Use a Hint — but you only get a limited number per mode, so use them wisely.
